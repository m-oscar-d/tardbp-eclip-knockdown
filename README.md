# Do TDP-43's direct RNA targets respond to knockdown?

TDP-43 (TARDBP) is the RNA-binding protein central to ALS. When knocked down, thousands of genes would change expression. However, many of those changes are indirect, meaning that changes in their expression are not the interest for the knowckdown of TDP-43. Therefore, this project aims to analyze whether the transcripts TDP-43 physically binds change more than the transcripts it does not. 

**Data:** ENCODE eCLIP (TARDBP, K562) and shRNA-knockdown RNA-seq (TARDBP, K562).
**Language:** R.
